# How to fold a paper airplane:

1. Get a plain sheet of paper.
2. Fold paper in half like a hotdog and then unfold.
3. Connect corners to middle line.
4. Repeat step 3 and apply double fold.
5. Fold paper airplane backwards on itself.
6. Unfold it.
7. Now fold it downward to finish the airplane.
8. Now you can throw it.
